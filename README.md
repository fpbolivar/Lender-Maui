# Lender - iOS Application

.NET MAUI application for iOS.

## Build

```bash
dotnet build Lender.csproj --framework net10.0-ios
```

## Run

```bash
dotnet run -f net10.0-ios
```

## Project Structure

- **Scripts/** - All app code (XAML, ViewModels, Services)
- **Lender.csproj** - Project configuration
- **MauiProgram.cs** - App startup
